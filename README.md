# My NEA project for A-level Computer Science

## To clone this project

1. Make a fork of the project
2. Install necessary dependencies
```
pnpm i
```
3. Create project build
```
pnpm build
```
4. Serve the completed build to the local 8080 port - alternatively you can choose the port yourself in the webpack config under `devServer.port`
```
pnpm start
```
